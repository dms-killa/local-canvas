# Forbidden Features List

The following features are strictly prohibited in the Local-Canvas project to prevent scope creep and hallucinations:

1. Cloud sync or cloud storage
2. Web search functionality
3. AI summarization (beyond basic text generation)
4. Autocomplete via external API
5. Online model updates or downloads
6. Remote file sharing
7. Browser-based interfaces or web frontends
8. External API calls (including Hugging Face, OpenAI, etc.)
9. Any functionality requiring internet connectivity
10. Features that would require data to leave the device

Any implementation of these features will result in immediate rejection and require rework.
